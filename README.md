### Simple algorithms
Algorithm | How it works
----      | ----
Swap (with temp variable) | To swap two variables x and y: **<br/><br/> - tmp = x; <br/> - x = y; <br/> - y = tmp;**
Swap (with arithmetic) | To swap two variables x and y using only addition and subtraction: **<br/><br/> - x = x + y; <br/> - y = x - y; <br/> - x = x - y;**
Swap (with XOR) | To swap two variables x and y using only XOR: **<br/><br/> - x = x XOR y; <br/> - y = x XOR y; <br/> - x = x XOR y;**
Factorial | A recursive function to compute the factorial of an integer. This is done as follows: **<br/><br/> int factorial(number): <br/> - if (number <= 1) return number; <br/> - else return number * factorial(number - 1);**
Fibonacci generator | A recursive function to compute the nth number in the Fibonacci sequence, as follows: **<br/><br/> int fibonacci(number): <br/> - if (number < 2) return number; <br/> - else return fib(number - 1) + fib(number - 2);**
